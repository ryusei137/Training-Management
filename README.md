# Training Management

## 概要

Pythonで作成したトレーニング重量管理プログラムです。

種目名と重量を管理し、登録されている種目の確認、追加、削除、重量変更、合計重量や最大重量などの集計を行います。

## 作成目的

日々のトレーニング記録を管理できるプログラムを作成することで、データ管理の考え方やPythonの辞書型の活用方法を学ぶことを目的として作成しました。

## 使用技術

* Python
* dictionary（辞書）
* if / elif / else
* for文
* input
* sum()
* max()
* min()
* len()

## 機能

* 種目一覧の表示
* 種目の追加
* 種目の削除
* 重量の変更
* 合計重量の表示
* 最大重量の表示
* 最小重量の表示
* 登録種目数の表示

## 工夫した点

辞書型を利用して種目名と重量を管理しました。

また、ユーザーの入力に応じてデータの追加・変更・削除ができるようにし、集計機能を実装することでデータ管理の基本を学習しました。

## 実行方法

python training.py
