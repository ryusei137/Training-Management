# Training Management

## 概要

Pythonで作成したトレーニング重量管理プログラムです。

種目名と重量を辞書型で管理し、登録済み種目の確認、種目の追加、削除、重量変更、各種データ集計を行うことができます。

## 作成目的

筋力トレーニングの記録を管理するプログラムを作成することで、Pythonの辞書型や繰り返し処理、条件分岐、データ集計の方法を学ぶことを目的として作成しました。

## 使用技術

* Python
* dictionary（辞書）
* input
* if / else
* for文
* f-string
* sum()
* max()
* min()
* len()

## 機能

* 登録済み種目の重量確認
* 新規種目の追加
* 種目一覧表示
* 合計重量の計算
* 最大重量の計算
* 最小重量の計算
* 平均重量の計算
* 最大重量種目の表示
* 種目数の表示
* 種目の削除
* 重量の変更

## 工夫した点

辞書型を利用して種目名と重量を管理しました。

また、ユーザーが入力した種目が登録されていない場合は、新しい種目を追加できるようにしました。

さらに、for文と組み込み関数（sum、max、min、len）を利用して、トレーニングデータの集計機能を実装しました。

最大重量を持つ種目を表示する機能も追加し、データ分析の要素を取り入れました。

## 実行方法

python training.py

## 実行例

種目名を入力してください: ベンチプレス

ベンチプレスの重量は80kgです

合計重量は300kg

最大重量は120kg

最小重量は80kg

種目数は3個

最大重量の種目はデッドリフトです

平均重量は100.0kgです
